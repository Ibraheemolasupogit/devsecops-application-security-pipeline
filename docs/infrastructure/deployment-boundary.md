# Deployment Boundary

Milestone 4 is configuration-only.

Implemented:

- Terraform modules and environment compositions.
- Local policy tests.
- Deterministic infrastructure evidence.
- Documentation and threat-model traceability.

Not implemented:

- `terraform apply`.
- AWS resource creation.
- Image publication.
- Route 53 or DNS.
- ACM certificate issuance.
- WAF rules.
- Cognito or external IdP integration.
- Scanner pipelines.
- Release gates.

No AWS resource should be assumed to exist from this milestone.
