# Roadmap

The broader portfolio roadmap may include:

1. Repository foundation and secure reference application. Delivered.
2. Secure design and threat modelling. Delivered.
3. Authentication and object-level authorisation. Delivered.
4. CI/CD engineering controls.
5. Automated security scanning.
6. AWS and Terraform security.
7. Vulnerability management and release gates.
8. Developer enablement and Security Champions.

Milestones 1, 2 and 3 are implemented in this repository state. Future milestones should update this document when they are actually delivered.
