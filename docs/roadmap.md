# Roadmap

The broader portfolio roadmap may include:

1. Repository foundation and secure reference application.
2. Secure design and threat modelling.
3. Authentication and object-level authorisation.
4. CI/CD engineering controls.
5. Automated security scanning.
6. AWS and Terraform security.
7. Vulnerability management and release gates.
8. Developer enablement and Security Champions.

Only Milestone 1 is implemented in this repository state. Future milestones should update this document when they are actually delivered.
