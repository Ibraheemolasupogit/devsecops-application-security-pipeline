# Milestone 2

Milestone 2 establishes the security architecture and threat-modelling foundation for the Genomic Research Access API.

Delivered:

- STRIDE-led threat model.
- Machine-readable inventories and registers.
- Control traceability from threat to requirement to verification evidence.
- Residual-risk register.
- Deterministic evidence generation and checksum verification.
- Generated Markdown reports.
- CI and local Makefile validation integration.

Explicitly not delivered:

- Production authentication.
- RBAC or object-level authorisation implementation.
- AWS resources.
- Terraform.
- AppSec scanners.
- Release gates.
- Vulnerability lifecycle workflows.
- Deployment.

The milestone analyses future cloud-native controls without implementing them.
