# Limitations And Residual Risk

The repository is portfolio-ready, not production-ready.

- The API is locally validated and not deployed.
- AWS resources are reference Terraform only.
- External identity provider integration is not live.
- Repository 5 integration is a contract export only.
- Findings remain under local governance with a `conditional_pass` release decision.
- Security Champions records are synthetic role-based records.
- No ticketing, dashboards, messaging or regulatory certification are included.
